COMMANDS:

Setup:
  1. ?conf show
  2. ?conf set {option} [what-to-set-it-to]
  3. ?conf reset {option}
  4. ?conf remove {option}
Moderation:
  • ?ban
  • ?kick
  • ?list
  • ?role
  • ?roleall
  • ?rollcall
  • ?purge
  • ?mute
  • ?unmute
  • ?warn
Miscellanious:
  • ?serverInfo
  • ?info
Standard:
  • ?accept
  • ?suggest
  • ?help
  • ?twitch
  • ?mixer
  • ?hug
  • ?roast
  • ?event (Disabled)
  • meme commands (use ?help for list of meme commands)
Streaming:
  • ?addMixer
  • ?addTwitch (Disabled)
  • ?delMixer
  • ?delTwitch (Disabled)
  • ?streamers
